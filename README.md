# Mood Journal 

Mood Journal is a desktop app that lets users log their daily mood, store entries in an SQLite database, and view mood trends over time on a chart. It also includes a calendar view for selecting days, optional notes for each entry, and quick stats like weekly, monthly, and overall average mood.
